Keyboard shortcuts
Key Action
i draft intentions for today
shift+i draft intentions for tomorrow
j/k next/previous intention in list
esc leave keyboard navigation (or, if inside 'add intention' box, leave that box)
space or x complete or un-complete intention
f nevermind or un-nevermind intention
a append text to intention
n edit notes
t go to today page
h go to history page
? show or hide keyboard shortcut list
