A to-do list based on simple principles

  1. You only add intentions for the current day.

    2. What do you want to achieve today?

  2. Start each day with a fresh list.

    3. Plan daily actions that feel alive. Intentions for past days go to your history.

  3. Stop organizing and start doing.

    4. No editing intentions. You can append text if new information arises.

    No deleting intentions. You can't un-intend to do something. No longer a priority? Mark it as 'Never mind'.
